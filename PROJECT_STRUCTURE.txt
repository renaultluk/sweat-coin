Quick Commands Reference
========================

Setup & Installation:
  npm install                    Install dependencies
  cp .env.example .env          Create environment file
  chmod +x quickstart.sh        Make script executable
  ./quickstart.sh               Run automated setup

Development:
  npm run compile               Compile contracts
  npm test                      Run tests
  npm run clean                 Clean build files

Local Deployment:
  npm run node                  Start local blockchain (Terminal 1)
  npm run deploy:local          Deploy to local (Terminal 2)

Testnet Deployment:
  npm run deploy:sepolia        Deploy to Sepolia testnet
  npm run deploy:polygon        Deploy to Polygon mainnet

Testing:
  npm test                      Run all tests
  REPORT_GAS=true npm test      Run tests with gas report
  npm run test:coverage         Run tests with coverage
